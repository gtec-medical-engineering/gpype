Paradigm Presenter

class gpype.frontend.widgets.paradigm_presenter.ParadigmPresenter(paradigm=None)[source]

Bases: Widget

Control widget for g.tec ParadigmPresenter integration.

Provides a user interface for loading, starting, and stopping paradigms using the g.tec ParadigmPresenter library. Supports both folder-based paradigm selection and manual file loading.

Parameters:

paradigm (str)

__init__(paradigm=None)[source]

Initialize the ParadigmPresenter control widget.

Parameters:

paradigm (str, optional) – Path to paradigm file (.xml) or folder containing paradigm files. If file, it will be loaded directly. If folder, a dropdown will be created for paradigm selection. If None, a file dialog will be used for loading.

terminate()[source]

Clean up resources when the widget is terminated.

Closes ParadigmPresenter windows and shuts down the presenter before calling the parent terminate method.
