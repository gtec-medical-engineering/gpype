class gpype.backend.filters.base.butterworth.Butterworth(fn, btype, order=None, **kwargs)[source]

Bases: IONode

Butterworth filter implementation for real-time signal processing.

This class implements a Butterworth digital filter using second-order sections (SOS) for improved numerical stability. It supports lowpass, highpass, bandpass, and bandstop filtering with configurable order.

The filter maintains internal state for continuous processing of streaming data, making it suitable for real-time applications in BCI and signal processing pipelines.

Variables:

  • DEFAULT_ORDER – Default filter order (2) when not specified.

  • _sos – Second-order sections representation of the filter.

  • _z – Filter state for each channel to maintain continuity between steps.

Parameters:

  • fn (list)

  • btype (str)

  • order (int)

__init__(fn, btype, order=None, **kwargs)[source]

Initialize the Butterworth filter with specified parameters.

Parameters:

  • fn (list) – List of cutoff frequencies in Hz. For lowpass/highpass: single value in list. For bandpass/bandstop: two values [low, high].

  • btype (str) – Filter type. Must be one of ‘lowpass’, ‘highpass’, ‘bandpass’, or ‘bandstop’.

  • order (int) – Filter order. Higher orders provide steeper rolloff but may introduce more phase distortion. Defaults to 2.

  • **kwargs – Additional arguments passed to parent IONode class.

Raises:

ValueError – If fn is not a list, btype is invalid, or order <= 0.
